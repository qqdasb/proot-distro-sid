# 安装

1.安装依赖

```bash
apt update && apt install wget
```

2.下载安装脚本

```bash
wget https://raw.staticdn.net/qqdasb/proot-distro-sid/main/debian-unstable.sh -O $PREFIX/etc/proot-distro/debian-unstable.sh
```

3.安装

```bash
pd install debian-unstable
```
